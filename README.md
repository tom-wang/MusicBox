# ssr_starter
## 1、克隆，安装依赖
```shell
git clone https://github.com/tom-wang/ssr_starter.git
npm install
```

## 2、构建
```shell
npm run build
npm run build:server
```

## 3、测试（CLI）
```shell
# 正常情况会输出一段html
node src/cli.js
```

## 4、测试（Web Server）
```shell
# 打开http://localhost:3000/查看
node src/server.js
```
